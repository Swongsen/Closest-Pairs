Compiler: MinGW gcc version 9.2.0
Files needed: main.cpp
To compile: g++ -o output.o main.cpp
To run: output.o

To use the sample tests, you can just copy and paste the input after running the program or enter the sequence manually.

Once the program is running, a menu of 4 options will be shown.
Option '1' is to manually input a point.
Option '2' is to automatically generate points.
Option '3' is to set a M value.
Option '4' is to calculate the M closest pairs.
The finished output will be written to "output.txt" as well as shown on the terminal.
